# `InterflexProvider`

The `interflex.ts` file exports an `InterflexProvider` component that can be used to wrap your application. It can be used instead of the `withInterflex` function generated by the CLI.

Using the `InterflexProvider` component can be helpful if you want to customise the behaviour of the `interflex` package.

Here is an example of how to use the `InterflexProvider` component:

```tsx filename="pages/_app.tsx"
import { useRouter } from "next/router";
import type { AppProps } from "next/app";
import { InterflexProvider, type I18nLanguage } from "~/i18n/interflex";

const App = ({ Component, pageProps }: AppProps) => {
  const router = useRouter();

  return (
    <InterflexProvider
      locale={router.locale as I18nLanguage}
      onChangeLocale={async (newLocale) => {
        // Do something when the locale changes.
        // ...

        await router.push(router.asPath, router.asPath, {
          locale: newLocale,
        });
      }}
    >
      <Component {...pageProps} />
    </InterflexProvider>
  );
};

export default App;
```
