# Interflex

Interflex is an easy-to-use, simple and lightweight library for managing your app's translations. It provides you with a convinient dashboard and a simple API to manage your i18n values.

## How does it work?

<Showcase />

## Main features

- **Typesafe everywhere** - Interflex is written in TypeScript and provides you with typesafe API. It also generates types for your translations based on the values downloaded from the web dashboard.
- **Simple npm package** - Interflex provides you with a simple API to manage your translations.
- **Web dashboard** - Interflex provides you with a convinient web dashboard to manage your translations. Don't edit your translations in JSON anymore!
